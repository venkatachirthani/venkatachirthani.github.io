Financial performance dashboards built in Power BI using SEC EDGAR filing data.

Costco Wholesale — Financial Performance Dashboard

Power BI SEC EDGAR API DAX

Costco's business model depends on volume and membership economics rather than high retail margins, which makes its financial statements read differently from a typical retailer. This project analyzes five years of Costco's SEC filings to answer a specific question: how does Costco sustain profitability while operating on razor-thin merchandise margins?

Using SEC EDGAR's XBRL API, I built an automated data pipeline in Power Query that pulls structured financial facts directly from Costco's 10-K and 10-Q filings, resolves inconsistencies in how financial concepts are tagged across filing years, and models the result in a proper star schema.

Starbucks — Financial Performance Dashboard

Power BI SEC EDGAR API DAX

Starbucks operates a fundamentally different economic model than a wholesale retailer — premium pricing, high gross margins, and a much more seasonal revenue pattern tied to holiday demand. This project uses the same EDGAR-based pipeline built for the Costco analysis to explore Starbucks' financials, letting the two dashboards serve as a direct comparison of two very different retail economics.

The dashboard highlights Starbucks' quarterly seasonality, tracks margin compression and recovery across recent years, and includes a balance sheet view that surfaces negative stockholders' equity caused by sustained share buybacks, with return-on-equity calculations adjusted accordingly.
